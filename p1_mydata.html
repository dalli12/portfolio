이통사 마이데이터 서비스 Flow






프로젝트 요약

마이데이터 서비스 정보 제공을 위해 이동통신사 고객의 실시간 청구, 납부, 결제데이터를 실시간으로 중계기관(KAIT)과 연동하는 AP 개발을 진행했습니다.

  • 개발내용 : 전용선을 통한 실시간 고객 빌링데이터 송수신
  • 기간 : 2021.08 - 2021.11


    • 인력구성 및 역할

  • 프로젝트 인원 : IT아웃소싱 2명(PL 1명, 개발자 1명), 유선사업부서 담당 1명
  • 데이터 연동 전문 검토 및 데이터 추출 SQL 작성 (기여도 80%)
  • 개발 일정 관리, 소스 배포 및 데이터 연동 테스트 (기여도 50%)


    • 상세 수행 내용

  • KAIT 와 고객 빌링 데이터 연동을 위한 전용선 및 방화벽 등 인프라 이슈 검토
  • 데이터 전문 규격 및 데이터 조회 기준 검토
  • 데이터 Request에 대한 빌링 데이터(청구/미납/결제) response 데이터 조회 SQL 작성
  • JAVA / Spring 기반 API 개발 및 테스트 (IT아웃소싱업체 개발 진행)

    • Tools Used

    JAVA
    SQL
