프로젝트 요약

현재 모든 학교에서 학생들에게 ‘소방안전교육’을 진행하고 있지만, 학교에서 제공하는 교육영상과 선생님의 지도로는 부족함이 있다 생각하였습니다. 게임의 재미 요소를 더하여 가상공간에서의 소방훈련 체험의 기회를 제공하고자 소방안전교육 프로젝트를 기획했습니다. AccuPS 업체로 부터 컨트롤러를 제공 받았고 해당 업체 실무자의 조언을 얻으며 프로젝트를 진행했습니다.

  • 개발내용 : VR 시뮬레이션 게임 개발
  • 기간 : 2017.06
  • 서비스 대상 : 초/중학생 대상


    • 인력구성 및 역할

  • 프로젝트 인원 : 4명(인천대 2명, 세종대 2명), 멘토 1명
  • 게임 진행 Scene 에 대한 미션 시나리오 및 사물 동작 구현 (기여도 60%)
  • 안드로이드 환경 기반 빌드 (기여도 50%)
  • Unity Object 디자인 및 게임 환경 구성 (기여도 30%)


    • 상세 수행 내용

  • Unity 기반으로 Scene 구성
    ➡️   시작화면, 튜토리얼, 게임진행, 종료화면 등 10개의 Scene 구성
  • 게임 Object 구성 및 동작 구현
    ➡️   학교 교실 환경으로 게임 이미지 구성
    ➡️   소화기, 엘레베이터, 화염 연기 등 Object 동작 구현
  • 게임 흐름 및 미션 구현 챗봇 솔루션 Persenalize
    ➡️   제한 시간, 미션 패널티, Sound 효과 구현
  • Google VR플랫폼 DayDream 을 통해 안드로이드로 연결 및 빌드
  • AccuPS 트레커/컨트롤러 제품과 연결

    • Tools Used

    Unity
    C#
